# 🔻 YIR Bottom Design

A modern bottom-style glassmorphism UI component for Flutter with blur effects, gradients, shadows, and fully customizable design — perfect for bottom sheets, navigation panels, and premium layouts.

---

## ✨ Features

🧊 Smooth glass blur effect
🎨 Custom color & gradient support
🌈 Modern bottom UI design
🌑 Shadow & elevation control
📐 Custom border radius & shape
👶 Supports any child widget
📱 Perfect for bottom sheets & nav bars
